## 开发环境
- :earth_asia: JDK 1.8
- :ghost: Vue 2
## 返回码状态
| 返回码 | 状态码含义 |
|  ----  | ----  |
| 200 | 请求成功 |
| 401 | 未经授权的访问或其他错误 |
## 优化
> 🥇 短信验证码改成邮件的方式获取验证码<br />
> 🥈 登录管理员之后用户端不需要登录，无法同时登录管理员和用户，无法做到独立，已解决<br />
> 🥉 用户端登录后可以直接输入后台地址到商家页面进行管理员操作，安全性低，已解决
